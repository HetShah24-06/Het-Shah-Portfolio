## 🚀 Deploy

### Vercel (Recommended — free, fast, instant)
```bash
# 1. Push this folder to a new GitHub repo
git init
git add .
git commit -m "feat: portfolio v1"
git remote add origin https://github.com/HetShah24-06/portfolio.git
git push -u origin main

# 2. Go to vercel.com → New Project → Import repo
# 3. Framework: Other (Static) — no config needed
# 4. Click Deploy ✅
```

### Netlify (Drag & Drop)
```
Drag the entire portfolio/ folder to netlify.com/drop
```

### GitHub Pages
```bash
# Rename repo to: HetShah24-06.github.io
# Settings → Pages → Source: main / root
```

## 🎨 Customisation

All design tokens live at the top of `css/style.css`:
```css
:root {
  --cyan:   #00ffc8;   /* Primary accent */
  --blue:   #00aaff;   /* Secondary accent */
  --purple: #9d4edd;   /* Tertiary / education */
  --bg:     #020408;   /* Page background */
  /* ... */
}
```
Change `--cyan` to any color to re-theme the entire site instantly.
